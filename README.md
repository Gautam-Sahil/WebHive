WebHive is a next-generation AI development platform that allows users to generate, visualize, and see web applications code in real-time. Unlike standard chatbots, WebHive spins up secure cloud sandboxes to run code live as it's being written.

## üõ†Ô∏è Tech Stack

Built with the **T3 Stack** ideology, leveraging the latest bleeding-edge technologies.

### Core Framework
- **[Next.js 16](https://nextjs.org/)** (App Router) - The latest React framework.
- **[React 19](https://react.dev/)** - Leveraging the newest concurrent features.
- **[TypeScript](https://www.typescriptlang.org/)** - Strict type safety throughout.

### AI & Execution Engine
- **[E2B Code Interpreter](https://e2b.dev/)** - Secure cloud sandboxes for live code execution.
- **[OpenRouter API](https://openrouter.ai/)** - Access to top-tier LLMs (Claude 3.5, Llama 3, mistral etc.).
- **[Inngest](https://www.inngest.com/)** (+ Agent Kit) - Durable execution for long-running AI workflows and queues.

### Backend & Data
- **[tRPC v11](https://trpc.io/)** - End-to-end typesafe APIs.
- **[Prisma](https://www.prisma.io/)** - Next-generation ORM.
- **[Neon](https://neon.tech/)** - Serverless PostgreSQL database.
- **TanStack Query** - Asynchronous state management.

### UI & Styling
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Utility-first CSS.
- **[Framer Motion](https://www.framer.com/motion/)** - Production-ready animations.
- **[Radix UI](https://www.radix-ui.com/)** - Unstyled, accessible UI primitives.
- **Sonner** - Toast notifications.
- **Lucide React** - Beautiful iconography.

### Authentication & Validation
- **[Clerk](https://clerk.com/)** - Complete user management.
- **Zod** - Schema validation.
